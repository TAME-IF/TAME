Quick Reference

Command Name Arguments Returns Description
NoOp None Nothing Does nothing.
QueueAction ACTION Nothing Queues a general action ACTION to be executed once this one completes.
QueueActionString ACTION, VALUE(STRING) Nothing Queues an open action ACTION with target VALUE to be executed once this one completes.
QueueActionObject ACTION, OBJECT Nothing Queues a transitive/ditransitive action ACTION with target OBJECT to be executed once this one completes.
QueueActionForObjectsIn ACTION, OBJECT_CONTAINER Nothing Queues a transitive/ditransitive action ACTION for each object in OBJECT_CONTAINER to be executed once this one completes.
QueueActionForTaggedObjectsIn ACTION, OBJECT_CONTAINER, VALUE(STRING) Nothing Queues a transitive/ditransitive action ACTION for each object in OBJECT_CONTAINER with tag VALUE to be executed once this one completes.
QueueActionObject2 ACTION, OBJECT1, OBJECT2 Nothing Queues a ditransitive action ACTION with targets OBJECT1 and OBJECT2 to be executed once this one completes.
AddCue VALUE(STRING), VALUE(STRING) Nothing Adds a VALUE cue to the response with content VALUE.
Text VALUE(STRING) Nothing Convenience for AddCue("text", VALUE)
Textln VALUE(STRING) Nothing Convenience for AddCue("text", VALUE + "\n")
Textf VALUE(STRING) Nothing Convenience for AddCue("textf", VALUE)
Textfln VALUE(STRING) Nothing Convenience for AddCue("textf", VALUE + "\n")
Pause None Nothing Convenience for AddCue("pause", "")
Wait VALUE(INTEGER) Nothing Convenience for AddCue("wait", VALUE)
Tip VALUE(STRING) Nothing Convenience for AddCue("tip", VALUE)
Info VALUE(STRING) Nothing Convenience for AddCue("info", VALUE)
AsBoolean VALUE VALUE(BOOLEAN) Returns the boolean value of VALUE.
AsInt VALUE VALUE(INTEGER) Returns the integer value of VALUE.
AsFloat VALUE VALUE(FLOAT) Returns the floating-point value of VALUE.
AsString VALUE VALUE(STRING) Returns the string value of VALUE.
StrLength VALUE(STRING) VALUE(INTEGER) Returns the amount of characters in the string representation of VALUE.
StrConcat VALUE1(STRING), VALUE2(STRING) VALUE(STRING) Returns the resultant string of sticking two strings together: VALUE1 and VALUE2.
StrReplace VALUE1(STRING), VALUE2(STRING), VALUE3(STRING) VALUE(STRING) Returns the resultant string of taking VALUE1 and replacing the first occurrance of VALUE2 with VALUE3.
StrReplacePattern VALUE1(STRING), VALUE2(STRING), VALUE3(STRING) VALUE(STRING) Returns the resultant string of taking VALUE1 and replacing the first occurrance of RegEx pattern VALUE2 with VALUE3.
StrReplacePatternAll VALUE1(STRING), VALUE2(STRING), VALUE3(STRING) VALUE(STRING) Returns the resultant string of taking VALUE1 and replacing every occurrance of RegEx pattern VALUE2 with VALUE3.
StrIndex VALUE1(STRING), VALUE2(STRING) VALUE(INTEGER) Returns the index of the first occurrance of VALUE2 in VALUE1, or -1 if not found.
StrLastIndex VALUE1(STRING), VALUE2(STRING) VALUE(INTEGER) Returns the index of the last occurrance of VALUE2 in VALUE1, or -1 if not found.
StrContains VALUE1(STRING), VALUE2(STRING) VALUE(BOOLEAN) Returns TRUE if VALUE2 is in VALUE1, or FALSE if not.
StrContainsPattern VALUE1(STRING), VALUE2(STRING) VALUE(BOOLEAN) Returns TRUE if RegEx VALUE2 matches a part of VALUE1, or FALSE if not.
StrContainsToken VALUE1(STRING), VALUE2(STRING) VALUE(BOOLEAN) Returns TRUE if token VALUE2 is in VALUE1 (case-insensitive), or FALSE if not.
