CallFrom ( ELEMENT, NAME );

Calls a procedure block from an arbitrary element context.

Parameters

ELEMENT Any Element (world, player, object, container, room) The origin element context to call a procedure from (can also be player or room, for current player or room).
NAME Value or Expression The name of the procedure to call.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

Calls a procedure with the same name as the value parameter in the same element as the provided element, or within that element's lineage (parent elements).

Like the name of the procedure, the value can be any type. The procedure called must have a name equal to the value.

If the procedure cannot be found, an ERROR cue saying so is added to the response.

Example

See Also

Call — Similar to CallFrom, but from the current context.
CallFromMaybe — Same as CallFrom, except it doesn't add an ERROR cue.

Technical Junk

CallFrom always searches backwards from the specified context despite a block existing either earlier or later in the element's lineage. It also uses strict equality to search for the matching procedure. It must be the same type (e.g. "3" and 3 are NOT equal). So, whatever works for the === operator.
