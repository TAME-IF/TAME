Call ( NAME );

Calls a procedure block in the current element context's lineage.

Parameters

NAME Value or Expression The name of the procedure to call.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

Calls a procedure with the same name as the value parameter. The procedure must be in the same element as the command, or within its lineage (parent elements).

Like the name of the procedure, the value can be any type. The procedure called must have a name equal to the value.

If the procedure cannot be found, an ERROR cue saying so is added to the response.

Example

See Also

CallFrom — Similar to Call, but from an arbitrary element's context.
CallMaybe — Same as Call, except it doesn't add an ERROR cue.
CallFromMaybe — Same as CallFrom, except it doesn't add an ERROR cue.

Technical Junk

Call always searches backwards from the origin context despite a block existing either earlier or later in an element's lineage. It also uses strict equality to search for the matching procedure. It must be the same type (e.g. "3" and 3 are NOT equal). So, whatever works for the === operator.
